## 已知限制

### 当前版本限制

1. **仅支持 amd64 架构**
   - 暂不支持 arm64 (M1/M2 Mac)
   - 计划在 v1.1.0 支持多架构

2. **无 Kubernetes 支持**
   - 仅提供 docker-compose
   - Helm Chart 计划中

3. **单节点部署**
   - 不支持 Swarm/K8s 编排
   - 依赖外部负载均衡

4. **镜像体积**
   - ~50MB 可进一步优化
   - 考虑 scratch 基础镜像

5. **无自动备份**
   - 数据卷备份需手动执行
   - 计划添加备份 sidecar

## 性能问题

### 待优化项

1. **镜像拉取慢**
   - 原因: 镜像体积
   - 缓解: 使用镜像加速器

2. **首次启动慢**
   - 原因: Zig 运行时初始化
   - 缓解: 预热容器
