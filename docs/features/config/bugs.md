## 潜在改进

### 1. 配置热更新

**描述**: 当前不支持运行时重新加载配置

**解决方案**: 使用文件监听（inotify/FSEvents）

**优先级**: 中

---

### 2. 配置加密

**描述**: 敏感配置以明文存储

**解决方案**: 支持 AES 加密的配置文件

**优先级**: 高

---

### 3. YAML 支持

**描述**: 仅支持 JSON 和 TOML

**解决方案**: 添加 YAML 解析器

**优先级**: 低
