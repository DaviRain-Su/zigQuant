## 已知限制

### 当前版本限制

1. **仅支持现代浏览器**
   - Chrome 80+
   - Firefox 78+
   - Safari 14+
   - Edge 80+
   - 不支持 IE

2. **移动端体验有限**
   - 响应式布局基本支持
   - 复杂表格在小屏幕显示不佳
   - 计划 v1.1.0 优化移动端

3. **无离线支持**
   - 需要网络连接
   - PWA 支持规划中

4. **单语言**
   - 仅支持中文
   - 国际化 (i18n) 规划中

5. **无键盘导航**
   - 缺少完整的键盘快捷键
   - 无障碍支持规划中

## 性能问题

### 待优化项

1. **初始加载慢**
   - 原因: 首屏资源较大
   - 缓解: 代码分割、懒加载

2. **大数据表格卡顿**
   - 原因: 渲染大量 DOM
   - 缓解: 虚拟滚动

3. **图表重渲染**
   - 原因: 频繁数据更新
   - 缓解: 节流、增量更新
