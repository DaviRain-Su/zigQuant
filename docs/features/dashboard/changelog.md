## 版本规范

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

### 版本格式

```
MAJOR.MINOR.PATCH

- MAJOR: 不兼容的界面变更
- MINOR: 向后兼容的新功能
- PATCH: 向后兼容的 bug 修复
```

### 变更类型

- **新增**: 新页面、新组件、新功能
- **变更**: 现有功能的变更
- **弃用**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: bug 修复
- **优化**: 性能优化
