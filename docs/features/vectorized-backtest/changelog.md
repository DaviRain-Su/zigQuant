# Vectorized Backtest 变更日志

本文档记录向量化回测模块的所有重要变更。

---

## [未发布] - v0.6.0

### 计划功能

- **SIMD 指标计算**: 使用 @Vector 类型加速 SMA/EMA/RSI 计算
- **内存映射加载**: mmap 高效加载大型数据文件
- **批量信号生成**: 一次性生成全部交易信号
- **批量订单模拟**: 批量执行订单模拟
- **标量回退**: 不支持 SIMD 的 CPU 自动回退

### 性能目标

- 回测速度: > 100,000 bars/s
- 内存效率: < 2x 数据大小
- SIMD 加速比: > 2x vs 标量

---

## 版本规范

- **Major**: 不兼容的 API 变更
- **Minor**: 向后兼容的功能新增
- **Patch**: 向后兼容的 bug 修复
